# B2CMiniProgram
微信小程序电商模版

## 模版内容：
轮播图，快捷导航，商品瀑布流列表，定位，搜索
商品详情，参数管理， 商品图片详情，图片浏览器
购物车
我的，地址管理

## 模块管理：

address                - 地址列表
goods-detail           - 商品详情
index                  - 首页
map                    - 店铺地址
my                     - 我的
new-address            - 新建地址
search-detial          - 搜结果
shop-cart              - 购物车
